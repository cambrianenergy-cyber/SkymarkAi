## 🔧 System Architecture

### How It All Works Together
```
User Input
    ↓
Content_Writer Agent (generates content)
    ↓
Learning System (tracks & analyzes)
    ↓
Brand Profile (learns preferences)
    ↓
Suggestion Engine (AI recommendations)
    ↓
Auto-Posting (if enabled)
    ↓
User Reviews & Approves
    ↓
Content Posted
```

### Data Storage
All data stored in Firestore:
- `content_generations/` - History of posts you created
- `content_suggestions/` - AI recommendations
- `brand_profiles/` - What the system learned about you
- `saved_content/` - Posts you saved for later
